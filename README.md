# weblistaya

Sitio web de weblistaya — plataforma de diseño web.

## Stack

- React 19
- TanStack Start
- TypeScript
- Tailwind CSS
- Cloudflare Pages

## Desarrollo local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Se deploya automáticamente en Cloudflare Pages desde la rama `main`.

También está sincronizado con Lovable.app para ediciones visuales.

Último deploy forzado: 2026-06-16.
